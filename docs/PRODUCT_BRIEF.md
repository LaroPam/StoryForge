# Product Brief

## Working names

- TaleForge
- Живая Книга
- StoryQuest
- Mythoria
- Нити Судьбы

## One-line description

A premium AI-powered interactive story adventure where users turn a single idea into a living illustrated story with choices, consequences, memory, inventory, NPC relationships, quests, and endings.

## Core promise

Write one idea. Enter a living story. Shape what happens next.

## What this product is

This is:

- an interactive AI story game
- a cinematic visual novel
- an AI game master
- an illustrated adventure
- a lightweight RPG
- a personal story generator with memory and consequences

## What this product is not

This is not:

- a chatbot
- a basic text generator
- a generic AI wrapper
- a SaaS dashboard
- a forum
- a novel editor
- a heavy DnD clone

## Target users

People who like:

- interactive fiction
- RPGs
- fantasy and sci-fi
- visual novels
- AI creativity tools
- personalized entertainment
- writing prompts
- story games
- atmospheric experiences

## MVP user journey

1. User opens the landing page.
2. User clicks `Start adventure`.
3. User enters a story idea.
4. User chooses genre, tone, visual style, length, and difficulty.
5. User creates or generates a protagonist.
6. User sees a story preview.
7. User starts the adventure.
8. User reads a scene with a cinematic image area.
9. User chooses an action or writes a custom action.
10. The mock engine generates the next scene.
11. World state updates.
12. User continues the story.

## Design direction

Premium cinematic dark interface.

Keywords:

- atmospheric
- elegant
- mysterious
- emotional
- polished
- immersive
- readable
- story-first
- game-like but not cluttered

Avoid:

- childish fantasy parchment UI
- generic SaaS dashboards
- plain chat screens
- overwhelming RPG stats
- raw AI wrapper feeling

## Early success metrics

Primary early metrics:

- user reaches 5+ story turns
- user reaches 10+ story turns
- user returns to continue a story
- user starts a second story
- user completes a short adventure

The MVP succeeds if users feel: `I want to continue this story.`
