# AI Adventure Platform

Working title: **TaleForge / Живая Книга / StoryQuest**

An AI-powered cinematic interactive story adventure platform.

Users write one idea, choose a genre and style, create a protagonist, and play through a living illustrated story with choices, consequences, memory, inventory, NPC relationships, quests, and endings.

## Product principle

This is **not a chatbot**.

It is a premium cinematic interactive story game:

- interactive book
- visual novel
- AI game master
- lightweight RPG
- illustrated adventure

## MVP goal

Prove that users want to continue their generated adventures.

The MVP should allow a user to:

1. Open a premium landing page.
2. Start story creation.
3. Enter a story idea.
4. Choose genre, tone, visual style, length, and difficulty.
5. Create or generate a protagonist.
6. See a story preview.
7. Start an adventure.
8. Read a cinematic scene.
9. Choose an action or write a custom action.
10. Receive the next scene through a mock story engine.
11. See world state changes.

## Implementation strategy

Start with mock providers and a working gameplay loop.

Do not implement real AI, real image generation, database persistence, authentication, or payments until the core UI and mock gameplay loop feel excellent.
